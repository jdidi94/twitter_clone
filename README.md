## Getting Started

### Prerequisites

  ```sh
  npm install npm@latest -g
  ```

  ### Installation
  
 1. Clone the repo
   ```sh
   git clone https://github.com/jdidi94/twitter_clone.git
   ```
2. Install NPM packages in the client and the server folders:
   ```sh
   npm install
   ```
   3. To run the server:
   ```sh
   npm start
   ```
4. To run the platform:
   ```sh
   npm start
   ```
